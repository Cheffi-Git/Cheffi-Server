## 개요
전국 맛집의 인기 리뷰를 한눈에 확인하고 지역별/취향별 인기 리뷰 확인할 수 있는 플랫폼 서비스

## 주요 기능

1. 지역별/태그별 인기 리뷰 조회
2. 사용자 팔로우/언팔로우 기능
3. 음식/지역별 리뷰 검색
4. 기본적 로그인/로그아웃 기능
5. 알림 기능
6. 쉐피코인/포인트 획득 기능
7. 리뷰 구매 기능
